¿Cuál es la relación entre BD y SGBD?
2
No existe relación entre ambos conceptos
SGBD es el software que permite la creación y manipulación de una BD
SGBD es el modelo que permite determinar cómo estructurar los datos en una BD
Ambos conceptos representan lo mismo
